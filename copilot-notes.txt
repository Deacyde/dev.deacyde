RULE #1 — BEFORE ANY WORK ON ANY MACHINE
-----------------------------------------
Always git pull before starting. Never assume the repo is current.

  deacyde.com repo:   git pull origin master
  deacyde-site repo:  git pull origin main

RULE #2 — AFTER FINISHING ON ANY MACHINE
------------------------------------------
Commit and push everything before switching machines.
For deacyde-site, push to BOTH branches:

  git checkout production && git merge main && git push origin production
  git checkout main && git push origin main
